#텍스트 스타일
#위치/형태 변형
#배경 스타일
#display: flex
  • position
    컨텐츠를 원하는 위치로 이동시켜주거나 배치 방식을 설정할 수 있다.
    position:relative를 제외하고 방향(top, left, right, bottom) 설정은 필수이며, 방향을 이용하여 좌표값을 가지게 된다.
    top, bottom은 함께 사용할 수 없다. left, right는 함께 사용할 수 없다.
    • position: static: [default]position값을 초기화할 수 있다.
    • position: relative : 현재 위치를 기준으로 위치를 벗어나 배치할 있다.
    • position: absolute : 부모의 position을 기준으로 위치를 설정할 수 있다.
      ※ 기준을 주고자 하는 부모에게 position이 없다면 position:relative를 설정해주어야 한다. 다른 속성값은 위치가 변경된다.
    • position: fixed : 브라우저를 기준으로 위치를 고정할 수 있다. 단, 부모에게 transform이 있다면 부모가 기준이 된다.
    • position: sticky : 부모 영역 안에서 해당 요소를 고정할 수 있다.
      ※ display:flex적용하고 자식에게 sticky를 사용할 경우 display:flex; align-item; flex-start 함께 사용해한다.
    header {
  width: 100%;
  position:fixed;
  top:0;
  left:0;
}

ul {position:relative;}
ul li {
  position:absolute; 
  top: 0; 
  left:0;
}
  • transform : 변형
    해당 요소의 형태를 변형시켜줄 수 있다.
    이동, 기울기, 회전, 크기변경을 진행할 수 있으며 transform이라는 속성안에 4개의 속성값으로 구분된다.
    변경의 값들은 x축과 y축 그리고 z축을 이용하여 좌표값으로 변형할 수 있으며, 단위는 px, %, 회전과 기울기는 deg를 이용하여 각도로 설정할 수 있다.
    • translate(10px, 20px) / translateX(50px) / traslateY(50%) :
    • rotate(45deg) / rotateX(45deg) / rotateY(-45deg) :
  • 가운데 배치 : margin: 0 auto편
    컨텐츠 가운데 배치하는 방법 중 margin:0 auto;라는 것이 존재한다. margin: 위아래 좌우값이 설정되는 것드로 위아래는 변경가능하나 좌우값이 auto로 설정되어 있어야 한다.
    그리고 해당요소에게 width값이 있어야 한다.

    해당 요소에게 width값이 있고 가운데 배치하고자 한다면 본인에게 margin:0 auto를 적용하여 배치할 수 있다.

    div {
  width: 1280px;
  margin: 100px auto;
}
  • 가운데 배치 : 인라인요소 / img태그편
    img를 사용한 요소를 가운데로 배치하고자 할 때는 img태그에게 width값을 적용해주는 방법도 있지만 width값을 사용하지 않고 부모 요소의 가운데로 배치하고자 할 때는 text-aling:cneter를 사용할 수 있다.
    text-aling:center는 본인이 아닌 부모에게 적용해주어야 한다.
    이는 img뿐만 아니라 인라인요소, display:inline-block를 사용한 요소를 가운데 배치하고자 할 때 사용할 수 있다.
    단, text-align:center는 본인이 아닌 부모에게 사용해야 적용되는점을 주의해야 한다. 
    ul {
    text-align:center;
}
ul li {
    display: inline-block;
}
  • 가운데 배치 : position편
    width값을 가진 요소를 가운데 배치할 때는 margin:0 auto를 주로 사용한다.
    하지만position:absolute나 position:fixed사용 시 margin: 0 auto를 이용하여 가운데 배치할 수 없다.
    position:absolute를 사용한다고 하였을 때 top: 50%, left: 50%를 사용하여 부모 영역에서 반만큼 이동되도록 설정해준다.
    그리고 transform을 이용하여 해당 요소가 가진 크기의 반만큼 이동되로고 해주어야 한다.
    공식처럼 사용되기 때문에 외워두는게 좋다.
    transform은 변형을 시켜주는 속성으로 속성값으로 이동인 translate를 사용해주어야 한다.
    • transform: translate(10px, 50px) : 오른쪽으로 10px, 위로 50px이동
    • transform: translateX(50px) : 오른쪽으로 50px 이동
    • transform: translateY(-50%) : 자신이 가진 높이의 반만큼 위로 이동
    • transform: trnaslate(-50%, -50%) => 해당 값을 사용한다.
    section {
    position:relative;
}
section div {
  width: 1280px;
  position: absolute;
  top:50%;
  left:50%; 
  transform: translate(-50%, -50%);
}
  • font-size
    텍스트 크기를 조절할 때 사용된다., 텍스트 크기는 모든 브라우저에서 16px을 기본으로 설정되어 있다. 단 h1 ~ h6는 예외로 둔다.
    div {font-size: 20px;}
  • font-style
    글자 스타일을 변경할 수 있다. 기본스타일, 이탤릭스타일, 기울어진 스타일로 설정할 수 있다.
    속성값은
    • normal : 기본
    • italic : 이탤릭
    • oblique : 기울기
    가 있다.
    div {font-style: italic;}
  • color
    텍스트 색상을 변경할 수 있다.
    색상 이름, HEX(#FFFFFF)값, rgb(255, 255, 255), rgba(255, 255, 255, 0.5), hsl등의 속성값으로 표현할 수 있다.
    div {color: #ffcc00;}
  • text-decoration
    텍스트에 밑줄, 취소선, 윗줄을 표시해줄 수 있다.
    • none : 없애기
    • underline : 밑줄
    • overline : 윗줄
    • line-through : 취소선
    div {
  text-decoration: underline;
}
  • letter-spacing / word-spacing
    letter-spacing : 글자 사이 간격 word-spacing : 단어 사이 간격
    단위는 px이나 % 사용할 수 있으며 수치는 음수, 양수 사용할 수 있다.
    div {letter-spacing: 5px;}
  • text-shadow
    글자 모양대로 그림자를 적용할 수 있다.
    text-shadow : 가로거리 세로거리 번짐 색상
    • 가로 거리 : 텍스트로부터 그림자 까지의 가로 거리(오른쪽)
    • 세로 거리 : 텍스트로부터 그림자 까지의 세로 거리(아래쪽)
    • 번짐 : 양수 값을 사용하면 모든 방향으로 퍼져나가 그림자가 크게 표시되며, 음수 값일 땐 모든 방향으로 축소되어 보인다. 기본 값은 0이다.
    • 색상 : 한 가지 색상, 혹은 공백으로 구분하여 여러 색상을 지정할 수 있다. 기본 값은 현재 글자 색이다.
    div {text-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);}
  • text-align
    글자의 정렬을 지정할 수 있다. 좌측 정렬을 기본으로 가운데 정렬, 우측 정렬, 양끝 정렬이 있다.
    글자 정렬을 주로 사용되지만 inline을 포함하고 있는 자식 요소에게 정렬을 시켜줄 수 있다.
    • start : 현재 텍스트 방향의 시작 위치에 문단 정렬
    • end : 현재 텍스트 방향의 끝 위치에 문단 정렬
    • left : 왼쪽에 맞추어 문단 정렬 (기본 값)
    • right : 오른쪽에 맞추어 문단 정렬
    • center : 가운데 맞추어 문단 정렬
    • justify : 양쪽에 맞추어 문단 정렬
    • match-parent : 부모 요소에 따라 문단 정렬
    div {text-align: center;}
  • text-justify
    양끝 정렬 시 글자 간격이 어색하게 벌어지는 것을 조절시켜줄 수 있다.
    • none : 정렬하지 않음
    • auto : 웹 브라우저에서 자동으로 지정
    • inter-word : 단어 사이의 공백을 조절해 정렬
    • distribute : 인접한 글자 사이의 공백을 똑같이 맞추어 정렬 
    
div {
  text-align: justify;
  text-justify: distribute;
}
  • text-transform
    영문에서 대소문자 변경할 수 있다.
    • none : 기본 값
    • capitalize : 첫 글자 대문자
    • uppercase : 모두 대문자
    • lowercase : 모두 소문자
    • full-width : 전각 문자 변환
    div {text-transform: capitalize;}
  • line-height
    글자 세로 줄 사이 간격을 조절해줄 수 있다.
    글자 크기가 20px일 때 line-height값을 40px로 설정하면 위 아래 각 각 10px의 여백이 생긴다.
    줄 간격이 글자 크기보다 작으면 글자가 겹쳐지게 된다.
    브라우저에선 1.2를 기본으로 가진다. 숫자를 적용할 경우 폰트사이즈에서 영향을 받는다.
    폰트사이트 * 1.2 => line-height값이 된다. 1.3 ~ 1.8값이 가독성이 좋다.
    • normal : 브라우저에서 정한 기본 값. 보통 1.2이다.
    • px : px 값으로 간격 지정
    • 숫자 : 글자 크기의 배수(글자 크기의 1.3 ~ 1.6 정도가 가독성이 좋음)
    • 백분율 : 글자의 몇 %인지 지정
    • inherit : 부모 요소의 속성 값을 상속 받아서 설정
    div {line-height: 1.5}
  • word-break
    글자 줄바꿈이 이루어질 때 글자 하나로 줄바꿈, 문자로 줄바꿈을 설정할 수 있다.
    기본값으로 국문은 한 글자로, 영문은 단언로 줄바꿈이 이루어진다.
    • normal : 기본 값으로 국문은 단어로 줄 바꿈이 이루어지고, 영문과 숫자는 공백을 기준으로 줄 바꿈이 이루어진다. 만약 띄워 쓰기가 없다면 줄 바꿈이 적용이 안되고 부모 영역을 벗어나게 된다.
    • break-all : 국문, 영문, 숫자 모두 글자 하나로 줄 바꿈이 적용된다.
    • keep-all : 단어를 기준으로 줄 바꿈이 적용된다. 하지만 영문, 숫자에서 띄워 쓰기 없다면 부모 영역을 벗어나게 된다.
    div {work-break : keep-all;}
  • overflow-wrap
    줄 바꿈 처리에서 word-break : keep-all을 사용하면 국문은 단어(공백)을 기준으로 줄바꿈이 이루어진다.
    영문인 경우 띄워쓰기가 발생되기 까지 부모 영역을 벗어나게 되는데 이러한 현상을 보완하기 위해 overflow-wrap을 사용할 수 있다.
    • normal : 기본적인 분리 지점에서 줄 바꿈이 발생된다.
    • anywhere : 긴 단어나 URL과 같이 분리할 수 없는 문자열인 경우 임의의 지점에서 줄 바꿈이 발생한다.
    • break-word : `anywhere` 과 동일하게 중단할 수 없는 문자열도 임의의 지점에서 줄 바꿈을 허용한다. ***min-content*** 고유 크기를 계산할 때 소프트 랩 기회를 고려하지 않는다.

    url주소처럼 긴 문장인 경우 overflow-wrap: break-word만 사용하게 되면 줄바꿈을 어색하게 처리하는 경우가 있다. word-break: break-all와 함께 사용하면 음절을 기준으로 자연스럽게 처리할 수 있다.

    div.en {word-break: break-all; overflow-wrap: break-word; }
  • writing-mode
    text-orientation : upright
    텍스트를 세로로 작성할 수 있다. 세로작성 시 오른쪽에서 왼쪽으로, 왼쪽에서 오른쪽으로 방향을 설정할 수 있다.
    • vertical-lr : 세로로 왼쪽에서 오른쪽으로 배치
    • vertical-rl : 세로로 오른쪽에서 왼쪽으로 배치

    text-orientation : upright영문인 경우 세로로 작성하다 보면 90도로 돌아간 모습으로 표시가 되는 경우가 있는데 이를 해결하기 위한 사용법으로 사용할 수 있다. 

    div.en {
    writing-mode : vertical-rl;
    text-orientation : upright;
}
  • 말줄임표 설정
    말줄이표를 사용할 경우 너비 값을 가져야하기 때문에 블록 요소로 작성하여야 한다.
    띄워스기 있을 대 너비를 자동으로 줄바꿈이 이루어지기 때문에 white-space: nowrap가 적용되어야 한다.
    📢 한 줄 말줄임표
    • overflow: hidden; // 너비를 넘어가면 안보이게
    • text-overflow: ellipsis; // 글자가 넘어가면 말줄임(...) 표시
    📢 여러 줄 말줄임표
    한 줄일 때와는 다르게 white-space: nowrap속성을 사용하지 않으며, overlfow, text-overflow는 사용된다.
    새롭게 display: -webkit-box, -webkit-line-clamp: 2, -webkit-box-orient: vertical 이 사용된다.
    • display: -webkit-box 속성은 해당 영역을 box 형태로 관리되도록 하며,
    • -webkit-line-clamp 속성은 영역 내의 컨텐츠의 최대 라인 수를 결정합니다.
    • -webkit-box-orient 속성은 영역 박스 내의 정렬을 가로로 배치할지 세로로 배치할지를 결정하는 속성으로,
    
div-01 {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
}

div-02 {
    display: -webkit-box;
    -webkit-line-clamp: 2;
    -webkit-box-orient: vertical;
    overflow: hidden;
    text-overflow: ellipsis;
}
